vue组件间的通信方式

发表于2022-05-18|更新于2022-05-19
|字数总计:752|阅读时长:2分钟|阅读量:

组件间通信有好多种，但主要还是利用以下几个方式

props 传递数据

适用于：父子组件传递通信

注意事项：
如果父组件给子组件间传递数据(函数)，本质是子组件给父组件传递数据
如果父组件给子组件传递数据(非函数)，本质是父组件给子组件传递数据

书写方式：3 种

小提示：路由的 props
书写形式：布尔值，对象，函数形式

自定义事件

适用于场景：子组件给父组件传递数据

$on与$emit

事件注意事项

事件：系统事件，cick，双击，鼠标系列等等
自定义事件
事件源、事件类型、事件回调

1)原生 DOM: button 可以绑定系统事件例如：click 事件

2)组件标签: evevnt1 可以绑定系统事件(不起作用，因为属于自定义事件) .native(可以把自定义事件变为原生 DOM 事件)

全局事件总线$bus

适用于场景：万能

pubsub.js 在 React 框架中使用较多(发布与订阅)

适用于场景：万能

Vuex

适用于场景：万能

插槽

适用于场景：父子组件通信—–(一般结构)
分类：
默认插槽、
具名插槽、
作用域插槽：子组件的数据来源于父组件，子组件是决定不了自身结构与外观

组件间通信高级

v-model[组件通信方式的一种]

v-model 他是 Vue 框架中指令，主要结合表单元素一起使用(文本框、复选、单选等)
主要作用是收集表单数据

可以通过 value 与 input 事件实现 v-model 功能

v-model 实现原理：value 与 input 事件实现的，而且还要注意 v-model 可以实现父子组件数据同步

属性修饰符[组件通信方式的一种]

可以实现父子组件数据同步

:money.sync 代表父组件给子组件传递一个 props[money] 给当前子组件绑定一个自定义事件(update:money)

$attrs与$listeners

组件实例的属性，可以获取父组件给子组件传递的 props 与自定义事件
v-on 不能用@进行替换 v-bind 不能用:替换

$children与$parent

ref 可以获取到某一个组件
$children组件实例的属性，可以获取当前组件的全部子组件[数组]
$parent 组件实例的属性，可以获取当前子组件的父组件，进而可以操作父组件的数据与方法

混入 mixin

如果项目中有很多结构类似功能，想到组件复用
如果项目中有很多组件 JS 业务逻辑相似，则考虑到 mixin[可以把多个组件 JS 部分重复，相似地方进行整理]
